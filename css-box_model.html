CSS - Box Model

CSS - Box Model
  1. Czym jest model pudełkowy CSS?

    2. Model pudełkowy (Box model) to "pudełko", które owija się wokół każdego elementu HTML. Wykorzystując go możemy sterować wyglądem i układem całej naszej strony. Model pudełkowy pozwala nam precyzyjnie określić wymiary elementu HTML na stronie. Każdy element, np. div i paragraf, można potraktować jak swego rodzaju prostokąt z określonymi wymiarami (możemy sami je określić lub przyjmą one wartość auto, czyli będą zajmować tyle miejsca ile potrzebują). Box Model składa się następująco z: zawartości "pudełka" (content box), marginesu wewnętrznego (padding), obramowania (border) i marginesu zewnętrznego (margin). Każdemu z tych pól możemy ustalić własne wartości, np. margin: 10px, border: 1px solid black, padding: 5px, width: 500px. Jeśli tego nie zrobimy przyjmą następujące wartości: margin: 0, border: 0 px black, padding: 0, width: auto. Gdyby nie Box Model strony internetowe wyglądałyby sztywno, każdy element byłby "przyklejony" do siebie, nie byłoby żadnych odstępów między elementami.

    Przykład:

    Tworzymy 3 divy na wzór strony blogowej: [Kliknij mnie] Ustawiamy margin: 0 i padding: 0 w body. oraz jakieś kolory dla naszych pudełek by je odróznić. Jak można zauważyć nasze pudełka są "przyklejone do siebie". Oprócz width nie mają żadnego elementu modelu pudełkowego.

    Dodając elementom właściwości box model, np. border, padding i margin możemy zmienić nasz wygląd strony nie do poznania, np. oddzielić treść blogu od stopki strony i nawigacji. [Kliknij mnie]

    Dodając to kolejne właściwości CSS możemy zrobić podstawową stronę blogową.

    W modelu pudełkowym wyróżniamy 4 elementy:

    • Content box - zawartość pudełka, w którym pojawia się, np. tekst i obrazy,
    • Border - obramowanie, które otacza zawartość,
    • Padding - określa odstęp zawartości od obramowania (jest przezroczysty),
    • margin - określa odstęp całego modelu pudełkowego od innego elementu (jest przezroczysty).

  4. Za co odpowiada parametr css box-sizing?

    5. Domyślnie model pudełkowy działa w taki sposób, że jeżeli dla pudełka o zadeklarowanej przez nas wysokości i szerokości (np. 200px) dodamy border (5px) i padding (5px) to nasz kwadrat, będzie miał:

    • szerokość = 220px, ponieważ musimy dodać 5px borderu i 5px paddingu z lewej oraz 5px borderu i 5px paddingu z prawej,
    • wysokość = 220px, ponieważ musimy dodać 5px borderu i 5px paddingu z góry oraz 5px borderu i 5px paddingu z dołu.

    Jest na to sposób:

    Aby nasz kwadrat miał szerokość i wysokość równą 200px musielibyśmy ustawić szerokość i wysokość na 180px (bo 200px -5px borderu -5px paddingu z lewej/góry oraz -5px borderu -5px paddingu z prawej/dołu).

    Jest jednak łatwiejsze rozwiązanie:

    Wystarczy, że dopiszemy: box-sizing: border-box (domyślnie jest: box-sizing: content-box). Własność ta powoduje, że padding i border będą automatycznie wliczane do ustalonej przez nas szerokości/wysokości.
